URL Endpoints List:

  1. GET: localhost:8080/movies (List of all movies)
  2. POST: localhost:8080/movies (Add a new movie to the list) (title paramater) JSON
  4. GET: localhost:8080/movies/:title (Find a specific movie by title) (title paramater) JSON
  5. GET: localhost:8080/genres (List of all genres)(genre name paramater) JSON
  6. GET: localhost:8080/genre/:name (A specific genre) JSON
  7. GET: localhost:8080/directors (List of all directors) JSON
  8. GET: localhost:8080/directors/name (A specific director) (director name paramater) JSON
  9. POST: localhost:8080/user (Add new user) (user name paramater) JSON
  10. GET: localhost:8080/user/:username (find user by username) JSON
  11. PUT: localhost:8080/users/:username (Edit user information) (username paramater) JSON
  12. Delete: localhost:8080/user/:username (Deregister) (username paramater) JSON
