URL Endpoints List:

  1. GET: localhost:8080/movies (List of all movies)
  2. POST: localhost:8080/movies (Add a new movie to the list) (title paramater) JSON
  3. DELETE: localhost:8080/movies (Delete a movie from the list) (title paramater JSON)
  4. GET: localhost:8080/movies/uniqueID (Find a specific movie by title) (title paramater) JSON
  5. GET: localhost:8080/genres (List of all genres)(genre name paramater) JSON
  6. GET: localhost:8080/genre/uniqueID (A specific genre) JSON
  7. GET: localhost:8080/directors (List of all directors) JSON
  8. GET: localhost:8080/directors/uniqueID (A specific director) (director name paramater) JSON
  9. POST: localhost:8080/user (Add new user) (user name paramater) JSON
  10. PUT: localhost:8080/user (Edit user password) (user password paramater) JSON
  11. Delete: localhost:8080/user (Deregister) (user name paramater) JSON
